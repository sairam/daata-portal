What does Daata solve?

Solves Dev Pain Points

Proxy/Replay HTTP requests for modifying from your service-to-service calls on Development or Staging environments

Solves "Where Do I Host?" Problems

Transfer data securely from any number of machines to Daata server to analyse/modify/sort data

Automagically Generate Graphs

Generate Graphs from data that is uploaded periodically to find trends.

Drag/Drop Metrics Into Dashboards

(1080p ready)

Drag/Drop graphs into dashboards and just type in a URL into your TVs to host get live data

Secure! For Your Eyes only!

Host it on your own infrastructure and intranet or the internet with https and secure credentials

Open Source

Made in Go. All the code is open source and open for development

Pain Points Solved

  • Versioned Documentation by Project for teams inside/across companies. (Google Docs just wont cut it)
  • Inspire your team by displaying Code Coverage on your Status Screen
  • Reviewing designs/HTML files prototypes made. When you need to showcase to your co-worker
  • Fetching information from database and portray your daily revenue realtime
  • Make Dashboards from Code Coverage or Order Totals generated earlier to make insightful graphs
  • URL Shortners to easily share/shout/remember URLs for sending reports
  • The Pastebin you always wanted
  • Fetch data from 10+ servers and process into information

Who is this aimed for?

The tool is aimed for small development teams who want to host any miscellaneous data.

The tool can be self hosted on any server in your intranet or the internet. It has been Tested on Linux, Mac OS X.

The use case of this service is to help solve problems and pain points for hundreds of thousands of developers, not necessiarly to scale(although we usually work on large scale problems).
